# AMIDOG VET

AMIDOG VET es un sistema de administración para una veterinaria, la cual ofrece
una API publica para clientes que deseen reservar una hora para consulta con sus
mascotas, y un ERP privado el cual le permite al dueño del local interactuar con un
registro persistente de información sobre clientes, reservas, mascotas,
productos, etc.
